1.The MBCG-EN dataset is original from the Instagram Influencer Dataset(download link: https://github.com/ksb2043/instagram_influencer_dataset)
2.We extract the post and comments from its post info files and conduct some clean steps on it.
3.The whole MBCG-EN dataset contains 3,659,607 samples with 20,079,183 comments.
4.Due to the memory size limitation of the supplementary material, we provide a samll part of the MBCG-EN dataset here.
